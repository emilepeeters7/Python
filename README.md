############ BROMMERLIJST ############

OPDRACHT
--------------------------------------------------------------------
Dit is een eenvoudige command-line applicatie geschreven in Python.
De bedoeling van dit project is om brommers te beheren met behulp 
van een database.

Dit project is als opdracht en toont het gebruik van:
- classes
- SQLite
- Configuratiebestanden
- git versiebeheer


FUNCTIONALITEITEN
--------------------------------------------------------------------
- Brommers toevoegen
- Brommers verwijderen
- Brommers wijzigen
- Zoeken naar brommers op naam of merk
- Overzicht tonen van de opgeslagen brommers
- Overzicht naar een tekst file printen


BEDOELING VAN DIT PROJECT
--------------------------------------------------------------------
Dit project is ontworpen voor een zaak die brommers verkoopt en een eenvoudig overzicht nodig heeft van:
- welke brommers nog in de winkel aanwezig zijn
- welke brommers verkocht / verwijderd zijn
- welke brommers nieuw zijn toegevoegd

De applicatie biedt:
Een duidelijk overzicht van alle brommers die op dit moment aanwezig zijn in de winkel.
De mogelijkheid om brommers toe te voegen wanneer er nieuwe brommers in de winkel komen. 
Brommers kunnen verwijderen uit de lijst als ze verkocht zijn of niet meer in de winkel aanwezig zijn.
Zoekfunctionaliteit (bijvoorbeeld op merk of naam) als klanten voor een specifieke brommer zoeken.
Het doel is om met een eenvoudige command line applicatie snel inzicht te krijgen in de voorraad.
